Shows auditing options.

sp_displayaudit [audcmd [, name]]

Parameter information :
»audcmd : Specify about what to show the auditing options. Possible values: database, default_object : Audit options for a new table or view in the named database., default_procedure : Show audit options for a new stored procedure or trigger in the named database., global : Show audit options for the named global option., login, object : Table or view., procedure : Stored procedure or trigger.
»name : The name related to the audcmd parameter.
