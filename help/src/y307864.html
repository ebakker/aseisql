Set a rule for device expansion

sp_dbextend "set","device", db_devname {[, arg1] [, arg2]}

Parameter information :
»db_devname : The name of a device or 'default' for a server wide setting.
»arg1 : The amount of space the device can grow by. You can use a percentage or a unit specifier (like "100M").
»arg2 : The maximum size of the device. No further expansion will occur when this size has been reached.
