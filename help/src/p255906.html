Show permission information for all columns in a table or view.

sp_table_privileges table_name [, table_owner [, table_qualifier]]

Parameter information :
»table_name : The name of a table or view.
»table_owner : The name of the owner of the table. When not specified the name of the current user is used and then dbo to look for the table.
»table_qualifier : The name of the database where the table if located. Can only be null or the name of the current database.
