Show settings and active sessions bound to a database.

sp_tempdb "show" [, arg1 [, arg2]]

Parameter information :
»arg1 : Choose the information you would like to see. Possible values: all : Show everything, app : Show bindings to an application., db : Show databases to group membership., gr : Bindings to a group., login : Bindings to a login name.
»arg2 : The name of database (arg1 must be db), login (arg1 must be "login") or application (arg1 must be "app").
