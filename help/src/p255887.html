Shows engine-group assignments for logons, stored procedures and applications.

sp_showcontrolinfo [object_type [, object_name [, spid]]]

Parameter information :
»object_type : Shows information for a specified type. Possible values: AP : Application, EG : Engine group, LG : Login, PR : Stored procedure, PS : Session.
»object_name : Name of the object or null when you specified "PS" as object_type.
»spid : When you specified "PS" as object type enter the session id, leave null for other types.
