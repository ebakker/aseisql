Returns the starting position of the first occurence of a pattern within a character expression.

patindex(pattern, char_expr [, using return_type])

Parameter information :
»pattern : The pattern you are looking for. Use any standard pattern match characters such as %.
»char_expr : A variable, constant or column of datatype char or varchar (null or not null) or of unichar or univarchar.
»return_type : Specifies the output type of the starting postion. Possible values: bytes : The offset is returned in bytes., characters : The offset is returned in characters., chars : The offset is returned in characters.
