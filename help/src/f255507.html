Returns a value for the combination of context and attribute name.

get_appcontext("context_name", "attribute_name")

Parameter information :
»context_name : The name of an application context. Datatype char(30). A predefined application context is sys_session.
»attribute_name : The attribute name for an application context. Datatype char(2048). For the predefined sys_session context the following attributes can be used Possible values: applname, character_set, clientapplname, clienthostname, clientname, current_dbid, current_dbname, current_groupid, current_userid, dateformat, hostname, is_ansi_null_on, is_noexec_on, is_showplan_on, language, lasterror, nesting_level, proxy_name, proxy_suserid, roles, rowcount, servername, session_groupid, session_userid, spid, transaction_count, username
