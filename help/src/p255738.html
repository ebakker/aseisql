Deletes result from the dbccdb database.

sp_dbcc_deletehistory [cutoffdate [, dbname]]

Parameter information :
»cutoffdate : The datetime value that is used to delete the results that are known for this date, or before this date. When no date is given only the most recent result is kept.
»dbname : The name of database. When not given all configured databases are used. You can also specify a dbid.
