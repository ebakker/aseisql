For the server

sp_setpglockpromote "server", null, new_lwm, new_hwm, new_pct

Parameter information :
»new_lwm : New value for the low watermark threshold, can be null to leave it unchanged.
»new_hwm : New value for the high watermark threshold, can be null to leave it unchanged.
»new_pct : New value for the lock promotion threshold, can be null to leave it unchanged.
