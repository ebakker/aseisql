Change the wash size.

sp_poolconfig cachename, parm1, parm2

Parameter information :
»cachename : Name of an existing cache.
»parm1 : IO size of the existing pool.
»parm2 : The new wash size as in the following format "wash=size". Where size is specified in multiples of the following: Possible values: G : GBytes, K : KBytes, M : MBytes, P : Pages
