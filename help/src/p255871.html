Renames an object or user defined datatype.

sp_rename objname, newname [, indorcol]

Parameter information :
»objname : Name of the table, column of the table, view, trigger, rule, default, constraint, index, stored procedure or user defined datatype.
»newname : The new name.
»indorcol : Use this to indicate if you want to rename an index or a column in case they have the same name. Possible values: column : Rename column, index : Rename index.
