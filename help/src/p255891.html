Allows the SQL text of a job to be manipulated or listed.

sp_sjobcmd name [, option [, text]]

Parameter information :
»name : The name or ID of a scheduled job or job. Possible values: jname, sjname
»option : Action to perform Possible values: add : Add a piece of SQL code to the job. You can use add several times to concatenate sql code., all_users, drop, list, sp_name
»text : The SQL text to store.
