Show information on stored procedures.

sp_stored_procedures [sp_name [, sp_owner [, sp_qualifier]]]

Parameter information :
»sp_name : Name of a stored procedure. Wildcards can be used.
»sp_owner : The name of the owner of the stored procedure. Wildcards can be used.
»sp_qualifier : The name of the database where the stored procedure is locaed. Can only be null or the name of the current database.
