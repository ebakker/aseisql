Returns the number of used pages for a table or index.

used_pgs(object_id, doampg, ioampg)

Parameter information :
»object_id : The ID of a table.
»doampg : The doampg value for the table (or clustered index). The value can be retrieved from sysindexes.
»ioampg : The ioampg value for a non-clustered index. The value can be retrieved from sysindexes.
