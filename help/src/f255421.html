Shows the name of the specified datepart of a given (small)datetime value.

datename(datepart, date_expression)

Parameter information :
»datepart : A portion of a datetime value. Possible values: day, dayofyear, dd : Day, dw : Weekday, dy : Day of year, hh : Hour, hour, mi : Minute, millisecond, minute, mm : Month, month, ms : Millisecond, qq : Quarter, quarter, second, ss : Second, week, weekday, wk : Week, year, yy : Year
»date_expression : An expression of the datatype smalldatetime or datetime.
