Execute a command on the machine running ASE.

xp_cmdshell cmdstr [, no_output] [, return_status]

Parameter information :
»cmdstr : Operating system command to be executed.
»no_output : Suppress any output.
»return_status : The return value for this exec can contain the following: 0 : successful execution, 1 error in execution. When this option is not specified the return value indicates if the spawning to execute the command succeeded.
