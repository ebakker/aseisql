Sets parameters for recovery of user databases.

sp_setsuspect_granularity [dbname [, susgran [, "read_only"]]]

Parameter information :
»dbname : Name of a user database.
»susgran : Sets the level of impact if corruption has beendetected by recovery. Possible values: database : Makes the whole database inaccessible by marking it as suspect when a corruption has been detected., page : Marks only the corrupt pages suspect and inaccessable, allows access to the database
»read_only : When set, sets the database to read-only mode if a corruption has been found.
