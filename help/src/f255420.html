Shows the difference between two dates in the specified datepart value.

datediff(datepart, date1, date2)

Parameter information :
»datepart : The portion of a datetime to show the difference in. Possible values: day, dayofyear, dd : Day, dw : Weekday, dy : Day of year, hh : Hour, hour, mi : Minute, millisecond, minute, mm : Month, month, ms : Millisecond, qq : Quarter, quarter, second, ss : Second, week, weekday, wk : Week, year, yy : Year
»date1 : The date to be subtracted from date2.
»date2 : A datetime expression or an expression that can be evaluated into a datetime value.
