Drops informations about keys for a table.

sp_dropkey keytype, tabname [, deptabname]

Parameter information :
»keytype : Type of key to drop. This depends on how ther key was originally defined. Possible values: common : Key was defined with sp_commonkey., foreign : Key was defined with sp_foreignkey., primary : Key was defined with sp_primarykey.
»tabname : Name of the table or view.
»deptabname : Name of the related table when you drop a common or foreign key.
