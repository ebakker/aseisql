Changes the characteristics of a login.

sp_modifylogin loginame, option, value

Parameter information :
»loginame : Name of a login. You can also specify "all overrides" to reset the password expiration, minimum password length or the maximum failed logins to system configured values, use a value of -1 in this case.
»option : Option to change. Possible values: "add default role" : Name of a role to be activated when logging in., "authenticate with" : Used to indicate if authentication is done trough ASE, LDAP, PAM or ANY., "drop default role" : Name of the role to be de-activated when logging in., "login script" : When the users logs in, the stored procedure as set in the value parameter is executed., "max failed logins" : The number of failed logins to allow before a login is locked., "min passwd length" : Minimum length of a password., "passwd expiration" : The password expiration interval in days., defdb : Default database., deflanguage : The name of the default language., fullname : Name of the user.
»value : Value for the option.
