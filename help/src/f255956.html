Returns 1 if a specified pointer to a text column is valid, otherwise a 0 is returned.

textvalid(column_name, textpointer)

Parameter information :
»column_name : The name of table and column name (quoted) in the form "table_name.column_name". The column name should be a text column.
»textpointer : A pointer to a text or image column. See also the textptr function.
