Creates a user defined datatype.

sp_addtype typename, phystype, nulltype

Parameter information :
»typename : Name of the user defined datatype.
»phystype : The definition of the user defined datatype in any standard server datatype, except timestamp.
»nulltype : Indicates whether or not the datatype should be used for identity purposes or to allow / disallow nulls. Possible values: "identity" : Use for identity purposes., "not null", nonull, null
