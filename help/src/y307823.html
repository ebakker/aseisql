Display / change config. params

sp_configure [configname [, configvalue]]

Parameter information :
»configname : Display configuration info for a specified group, a specified configuration parameter name or for the names starting with the specified characters.
»configvalue : Change the specified configuration parameter (configname) to the specified value. The value can contain a unit specifier like p (pages), m (megabytes), g (gigabytes). Example: "2k".
