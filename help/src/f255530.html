Returns a specified expression when another expression evaluates to null.

isnull(expression1, expression2)

Parameter information :
»expression1 : The expression that is checked whether or not it is null. When it is null, expression2 is returned, otherwise expression1. Can a constant, column name, variable.
»expression2 : The expression that is returned when expression1 evalutes to null. Can be a constant, column name, variable.
