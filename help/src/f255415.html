Shows the number of free pages within a disk device fragment.

curunreservedpgs(dbid, lstart, unreservedpgs)

Parameter information :
»dbid : The internal id of a database.
»lstart : Any pagenumber within the database fragment. Query master.dbo.sysusages for values.
»unreservedpgs : The value to return if the function has been called with invalid parameters or when the database is not available.
