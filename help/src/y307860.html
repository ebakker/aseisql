Modify the rule for a device

sp_dbextend "modify","device", db_devname, arg1, arg2

Parameter information :
»db_devname : The name of a device for which you want to change the rule or 'default' for a system wide setting.
»arg1 : Indicates the value you like to change. Possible values: growby, maxsize
»arg2 : The new value for the rule.
