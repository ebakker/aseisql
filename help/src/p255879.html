Show and set options for server defined on the local server.

sp_serveroption [server, optname, optvalue]

Parameter information :
»server : Name of a remote server.
»optname : Option for a server. Possible values: "external engine auto start" : Starts the EJB server upon start of the ASE server., "mutual authentification" : Checkk the credentials of local and remote server (only for rpc security model B)., "net password encryption" : When true passwords will be sent encrypted to the remote server., "readonly" : When true the server is available for read only access (CIS only), "rpc security model A" : When true it uses a fairly low level of security., "rpc security model B" : Use a secure connection between local and remote server., "security mechanism" : When you use rpc security model B, this option must be set to true., "server cost" : (CIS only) Specifies the cost of a single exchange under the user's control, on a per-server basis., "server login" : (CIS only) Configure for server-server connection through the site handler., "timeouts" : When true physical connections timeout after 1 minute of no logical connection., "use message confidentiality" : Messages to, and results from, the remote server are encrypted (only for securty model B)., "use message integrity" : Check for tampering with the messages (only for security model B).
»optvalue : Value for optname. Possible values: false, true
