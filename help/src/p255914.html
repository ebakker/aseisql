Unbinds an execution class previously bound to a application, login or stored procedue.

sp_unbindexeclass object_name, object_type, scope

Parameter information :
»object_name : Name of the application, login or stored procedure bound to an execution class.
»object_type : Type of object_name. Possible values: ap : Application, lg : Login, pr : Stored procedure
»scope : The name of an application or login when object type is ap or lg, or the name of the owner of the stored procedure when type is pr.
