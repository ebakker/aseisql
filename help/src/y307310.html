Convert a (small)money datatype to char and set a display format

convert([var]char [(length)] [null | not null], expression ,style)

Parameter information :
»length : Length of the (var)char. 30 is default.
»expression : The (small)money expression to convert.
»style : Apply a style. Possible values: 0 : No formatting., 1 : Display a comma after every 3 digits.
