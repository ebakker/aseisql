Set a default location on the local server for objects residing on a remote server.

sp_defaultloc dbname, defaultloc [, defaulttype]

Parameter information :
»dbname : Name of the database on the local server.
»defaultloc : The specification of the remote location in the form "server_name.db_name.owner_name.". When you specify null at this point any existing definition is cleared.
»defaulttype : The type of objects to be mapped. Possible values: rpc, table, view
