Returns 1 for an enabled license feature, or 0 for a disabled or invalid license name.

license_enabled("license_name")

Parameter information :
»license_name : The name of a license feature. Possible values: ase_asm : Advanced Security Mechanism, ase_dev : Developer's Edition, ase_dtm : Distributed Transaction Management, ase_ha : High Availability, ase_java : Java, ase_sbe : Small Business Edition, ase_server : ASE itself.
