Unbinds a rule from a user defined datatype or a table column.

sp_unbindrule objname [, futureonly [, whichrule]]

Parameter information :
»objname : Name of a user defined datatype or the table and column name.
»futureonly : When specified the rule is unbound from the user defined datatype but kept on the columns already having this rule through the datatype.
»whichrule : Specify which rule should be bound / unbound. Possible values: accessrule : Indicates that you are unbinding the access rule bound to objname., all : All specifies that you are unbinding all rules bound to objname.
