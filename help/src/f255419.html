Adds the fragment of a date to a (small)datetime value.

dateadd(datepart, integer_expression, date_expression)

Parameter information :
»datepart : The portion of a date to add (or subtract). Possible values: day, dayofyear, dd : Day, dw : Weekday, dy : Day of year, hh : Hour, hour, mi : Minute, millisecond, minute, mm : Month, month, ms : Millisecond, qq : Quarter, quarter, second, ss : Second, week, weekday, wk : Week, year, yy : Year
»integer_expression : The number of time the datepart should be added.
»date_expression : The (small)datetime expression.
