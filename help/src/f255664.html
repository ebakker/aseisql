Set an application context value.

set_appcontext("context_name", "attribute_name", "attribute_value")

Parameter information :
»context_name : The name of an application context. Datatype char(30). A predefined application context is sys_session.
»attribute_name : The attribute name for an application context. Datatype char(2048).
»attribute_value : The value to be stored for the combination of context and attribute name.
