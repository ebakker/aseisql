Changes or shows the size for i/o's to the transactionlog.

sp_logiosize ["size"]

Parameter information :
»size : Sets or shows the size. Specify a value of units of the logical page size (maximum 4 times). Possible values: all : Display the size of io's for the transaction log for all databases., default : Use 4 Kb if a memory pool is availabe or use 2 Kb.
