Get the next message from mailbox. Windows NT only.

xp_findnextmsg msg_id output [, type] [,unread_only]

Parameter information :
»msg_id : The message identifier.
»type : The message type. Possible values: CMC:IPM, null : Defaults to "CMC:IPM"
»unread_only : Retrieve only unread messages. Possible values: false, true
