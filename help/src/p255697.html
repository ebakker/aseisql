Defines a server.

sp_addserver lname [,class [, pname]]

Parameter information :
»lname : Name of the server.
»class : Type of server. Possible values: access_server, ASAnywhere : Adaptive Server Anywhere version 6.0 or later., ASEJB : EJB server, ASEnterprise : All versions of Adaptive Server Enterprise; support for SQL Server 4.9 is not provided., ASIQ : Adaptive Server IQ of 12.0 or later., db2, direct_connect, local : Use local to set the name of the server you are connected to., null, RPCServer, sds : Conforms to the interface requirements of a Specialty Data Store as described in the Adaptive Server Specialty Data Store Developer's Kit manual., sql_server
»pname : Name of the server in the interfaces file, or a network based address in the format "hostname:portnumber" or "ip address:portnumber"
