Display information about cursors for your session.

sp_cursorinfo [cursor_level [, cursor_name]]

Parameter information :
»cursor_level : Nesting level within a stored procedure. Possible values: -1 : Show cursor information regardless of stored procedure nesting level., 0 : For cursors declared outside a stored procedure., n : Display information for the specified procedure nesting level n., null : Any level. Specify cursor_name too.
»cursor_name : Name of the cursor.
