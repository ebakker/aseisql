Defines an interface for controlling running jobs.

sp_sjobcontrol name, option

Parameter information :
»name : The name or ID of the scheduled job, a job, or a runid to be controlled.
»option : Specifies a comma-separated list of options and option_name=option_value pairs that define the action to perform. Possible values: all_users : allows users with the role js_admin_role to administer scheduled jobs owned by other users., disable : instances of the scheduled job with the given name or ID are disabled. Any currently running jobs are unaffected., enable : instances of the scheduled job with the given name or ID are enabled., run_now : schedules the job with the given name or ID to run now., start_js : starts Job Scheduler., stop_js : stops Job Scheduler and terminates any running jobs., stop_js_timeout : waits a specified number of minutes then stops Job Scheduler, terminating any running jobs., stop_js_wait : waits for running jobs to complete, then stops Job Scheduler., terminate : instances of the job with the given name, ID, or runid are terminated.
