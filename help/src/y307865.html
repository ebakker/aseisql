Set a threshold for a database and segment

sp_dbextend "set", "threshold", db_devname, arg1, arg2

Parameter information :
»db_devname : The name of a database or 'default' for a server wide setting.
»arg1 : Specify the name of a segment.
»arg2 : The amount of free space. You can use values like "2K", "0.5m" or just a number for kilobytes.
