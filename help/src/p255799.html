Makes a database accessable again after is has been marked suspect by recovery.

sp_forceonline_db dbname, option

Parameter information :
»dbname : Name of the database.
»option : Provides access to the database for particular users. Possible values: all_users : Provides access to all users., sa_off : Revokes access from users previously given with sa_on., sa_on : Allows only access for users who have sa_role.
