Set some table / index attributes

sp_chgattribute objname, optname, optvalue

Parameter information :
»objname : Name of the table or index.
»optname : Name of the option. Possible values: "concurrency_opt_threshold", "dealloc_first_txtpg" : Deallocates the first text or image page when it is no longer needed. Possible values for optvalue are 0 and 1., "exp_row_size", "fillfactor", "identity_gap" : Can only be set for tables, "max_rows_per_page" : Sets the number of rows per page for future allocations., "optimistic_index_lock" : Optimistic index locking can resolve increased contention on some important resources, such as the spinlocks that guard address locks on the root page of an index. Possible values for optvalue are 1 (on) and 0 (off)., "reservepagegap"
»optvalue : The value.
